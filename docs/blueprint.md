# **App Name**: Adygea Explorer

## Core Features:

- Interactive Map: Display an interactive map of Adygea with key locations and attractions.
- Route Planning: Enable users to plan custom routes with multiple destinations, considering time and distance. Save planned routes to local storage.
- Form Builder: Contact and booking forms submission including Name, Email, Phone Number, Message, Route selection, Travel date and number of people.
- Email Service: Configure email sending capabilities when the forms are submitted, making sure to deliver emails and validate them.
- Image Enhancement: Enhance gallery images automatically before display, powered by AI. Includes options for sharpening, noise reduction, and color correction for optimal viewing experience.
- AI Tour Guide: Use AI to provide customized tour guide tool giving users instant insights on location histories, local legends and relevant travel tips using their current map location. 

## Style Guidelines:

- Background color: Light beige (#F5F5DC) for a natural, inviting feel.
- Primary color: Turquoise-green (#40E0D0) to evoke the rivers and nature of Adygea.
- Accent color: Graphite (#808080) to add a touch of stone and earthiness. Text should be dark grey, not black.
- Headline font: 'Playfair' a serif font, with a classical, elegant style, well-suited for titles. Note: currently only Google Fonts are supported.
- Body font: 'Inter', a sans-serif font with a clean and readable style. Note: currently only Google Fonts are supported.
- Use minimalist line icons for navigation and key features to maintain a clean and modern look.
- Implement subtle fade-in, slide-up, and smooth hover animations (250-350ms, cubic-bezier) to enhance user experience without being overwhelming.
